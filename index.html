About this project

This document is a minimal version of the Dungeons & Dragons basic rules v0.3 website with the remaining information from SYSTEMS REFERENCE DOCUMENT (SRD) v5.1 to fill the gaps. The main idea of this project is to make a better version for tablets, ebooks or printing. Additonally we will be able to improve the document's references, contents or examples.
